Companion code to [What is it like to be a fly?](https://shunyaresearch.systems/blogs/fly.html)

**Requires:** `flygym[examples]`, `neuprint-python`

**neuPrint token:** Register at [neuprint.janelia.org](https://neuprint.janelia.org), sign in, and copy your token from Account > Auth Token. Set it in the notebook where `Client(...)` is called.

**Outputs:** `three_brains.png`, `exp_naive.mp4`, `exp_appetitive.mp4`, `exp_aversive.mp4`

More short experiments will be documented in this repository.
